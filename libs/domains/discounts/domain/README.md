# @panary/discounts/domain

Domänen-Schemas und reine Anwendungs-/Eligibility-Logik für Rabatte (Discounts).

- `discount.schema.ts` — TypeBox-Schema der Rabatt-Definition (Regel) + abgeleitete Data/Patch/Query-Schemas und Enums.
- `discount-apply.ts` — reine Funktionen zur Rabatt-Anwendung und -Eligibility (von Edge, Cloud und Frontend nutzbar).

Importpfad: `@panary/discounts/domain`.
