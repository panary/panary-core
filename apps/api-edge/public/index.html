Welcome to the Panary Core API

This is the official API for our project. Use this API to interact with our database and retrieve or manipulate data.

API-Version

Current version: 1.0.0

Authentication

Our API uses JSON Web Tokens (JWT) for authentication. To access protected endpoints, you must log in and obtain a token.

Unterstützte Authentifizierungsmethoden:

  • Local authentication: Login with email and password.

Details on authentication can be found in the API-Documentation.

Sample inquiries

Registration


POST /authentication
Content-Type: application/json

{
  "strategy": "local",
  "email": "benutzer@example.com",
  "password": "IhrPasswort"
}

Response:


{
  "accessToken": "IhrJWTToken",
  "user": {
    "id": "1234567890",
    "email": "benutzer@example.com"
  }
}

Access protected resource


GET /users
Authorization: Bearer IhrJWTToken

Antwort:


[
  {
    "id": "1",
    "text": "Hallo Welt!",
    "userId": "1234567890"
  },
  {
    "id": "2",
    "text": "Willkommen bei unserer API.",
    "userId": "0987654321"
  }
]
